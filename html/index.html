Dojow Server index.html

はじめに

Dojow server は Crystal で作られた小さなウェブサーバである。大規模アプリケーション向けではないので、家庭内 LAN などの環境で動作することを前提にしている。


インストール

GitHub から次のコマンドで clone する。

git clone https://github.com/makandat/dojow


ビルド

dojow フォルダ内の build.sh を ./dojow にて実行する。(Linux での使用を前提にしている)


実行

dojow フォルダ内で "./bin/dojow -v" を実行する。-v は verbose の意味でこれを付けないとリクエスト状態は表示されない。
